# 1. Purpose

This glossary provides standard definitions for business, technical, and agricultural terms used throughout the BhoomiAI project.

Using a common vocabulary ensures consistency across documentation, design, development, testing, and user training.

---

# 2. Business Terms

| Term | Definition |
|------|------------|
| Organization | The highest-level business entity using BhoomiAI. |
| Farm | A managed agricultural property within an organization. |
| Block | A subdivision of a farm containing one or more fields. |
| Field | A physical cultivation area where crops are grown. |
| Plot | A smaller section of a field used for planning or experimentation. |
| Crop Cycle | The complete lifecycle of a crop from planting to harvest. |
| Livestock | Animals managed within the farm, such as goats. |
| Batch | A group of items managed together, such as silkworm batches. |
| Inventory | Materials, supplies, and products managed by the farm. |
| Asset | Equipment, machinery, buildings, or other long-term resources. |

---

# 3. Operational Terms

| Term | Definition |
|------|------------|
| Task | A unit of work assigned to a user or team. |
| Workflow | A sequence of tasks following defined business rules. |
| Schedule | Planned timing for activities. |
| Activity | A specific farm operation, such as irrigation or vaccination. |
| Checklist | A predefined list of steps for completing a task. |
| Notification | A message generated by the system for users. |
| Approval | Formal confirmation required before an action proceeds. |

---

# 4. Technical Terms

| Term | Definition |
|------|------------|
| API | Application Programming Interface used for communication between systems. |
| Entity | A business object stored and managed by the system. |
| Module | A functional area of the platform, such as Crop or Inventory. |
| Service | Backend logic responsible for business operations. |
| Repository | Component responsible for data access. |
| Event | A recorded business action that may trigger automation. |
| Audit Log | A record of significant system activities. |

---

# 5. AI Terms

| Term | Definition |
|------|------------|
| AI Agent | A specialized AI component that performs a defined business role. |
| Prompt | Instructions provided to an AI model. |
| Knowledge Base | Structured information used by AI to answer questions. |
| RAG | Retrieval-Augmented Generation, where AI retrieves relevant information before generating a response. |
| Context | Information provided to AI to improve relevance and accuracy. |
| Tool | A capability available to an AI agent, such as searching records or creating tasks. |

---

# 6. Security Terms

| Term | Definition |
|------|------------|
| Authentication | Verifying a user's identity. |
| Authorization | Determining what a user is allowed to do. |
| Role | A collection of permissions assigned to users. |
| Permission | The right to perform a specific action. |
| Session | A user's authenticated interaction with the platform. |
